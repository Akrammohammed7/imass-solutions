# iMASS Solutions
### Consulting & Recruitment Management Platform  

Developed during a Remote Software Developer Internship at  
**LimrahSoft IT Solution Private Limited — Hyderabad, India**

---

## Overview

iMASS Solutions is a full-stack Consulting and Recruitment Management Platform designed to streamline consulting opportunities, job postings, candidate applications, and administrative workflows.

The system centralizes recruitment and consulting operations into a secure, role-based web application, demonstrating backend engineering, relational database design, authentication systems, and workflow automation.

---

## Project Impact

- Managed **25+ consulting job postings**
- Processed **100+ candidate applications**
- Reduced manual applicant tracking time by **60%**
- Improved administrative workflow efficiency by **40%**
- Automated resume uploads and candidate tracking processes

---

## Technical Stack

- **Backend:** PHP  
- **Frontend:** HTML, CSS, JavaScript  
- **Database:** MySQL  
- **Version Control:** Git & GitHub  

---

## Core Features

- Role-based authentication and access control
- Consulting and job opportunity management
- Candidate application tracking system
- Secure resume upload functionality
- Admin dashboard for centralized management
- Dynamic job search and filtering logic
- CRUD operations for job postings and applications

---

## Engineering Principles Applied

- Object-Oriented Programming (OOP) concepts
- Relational Database Design (Normalization – 3NF)
- Prepared SQL statements to prevent SQL injection
- Session-based authentication and authorization
- Data structures for dynamic query filtering
- Backend debugging and workflow optimization
- Software Development Lifecycle (SDLC) practices
